# 🎓 都立数学 完全特訓プレミアム
**〜 都立高校入試 数学 共通問題 徹底攻略アプリ 〜**

このアプリは、都立高校入試の数学（共通問題）に特化した、iPad/スマホで学べるインタラクティブ学習ツールです。

## 🌟 特徴
- **動くシミュレーター**: 「規則性」や「関数」をスライダーで動かして、視覚的に理解できます。
- **3D空間図形**: 空間図形を360度くるくる回して、裏側や断面を確認できます。
- **全大問網羅**: 第1問の小問集合から、第5問の空間図形まで、入試の全範囲をカバー。
- **重複防止システム**: 常に新しい問題にチャレンジできるよう、ランダム出題機能を搭載。

## 📖 コンテンツ内容
- **第1問：計算・小問集合** (46点分) - 基礎を完璧にする9つのジャンル
- **第2問：文字式の規則性** - タイルやマッチ棒の動きをシミュレート
- **第3問：一次関数** - グラフの動きと面積の関係を体感
- **第4問：平面図形・証明** - 図形を動かしながら合同・相似を理解
- **第5問：空間図形** - 3Dモデルで立体の構造をマスター

## 🛠️ 技術スタック
- **HTML5 / CSS3** (レスポンシブデザイン)
- **JavaScript** (ES6+)
- **JSXGraph** (幾何・関数シミュレーション)
- **Three.js** (3Dレンダリング)
- **MathJax** (美麗な数式表示)
