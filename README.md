# 都立数学特訓アプリ：大問1マスター

都立高校入試の数学「大問1（小問集合）」に特化した、Webブラウザで動く学習アプリです。

## 🚀 特徴
- **全90問搭載**: 計算から作図まで、都立入試の傾向に合わせた問題を網羅。
- **スマホ対応**: 通学中のスキマ時間にサクサク解けるUI。
- **丁寧な解説**: 全問題にステップごとの計算過程を記載。
- **インストール不要**: URLにアクセスするだけで今すぐ学習開始。

## 🛠 使い方
1. 学習したいカテゴリ（四則計算、連立方程式など）のボタンをタップ。
2. 問題が表示されたらノートに解く。
3. 「答え合わせ」ボタンで解説を確認。
4. 「次の問題」でランダムに新しい問題が出題されます。

## 📚 使用技術
- HTML / CSS / JavaScript
- [JSXGraph](https://jsxgraph.org/): 図形・作図の描画
- [MathJax](https://www.mathjax.org/): 数式の表示
